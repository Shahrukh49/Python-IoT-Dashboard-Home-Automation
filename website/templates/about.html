About Project

This project was developed by me using Python-Flask Webframework as backend, front end uses HTML with Bootstrap. The project can be taken as a template and can be used to make a bigger IoT Dashboard. API request is in json format which can be viewed at localhost:5000/api_key=shahrukh123 after the project is up and running. The request can be parsed by a microcontroller such as an ESP8266 or ESP32 which then can be used to turn appliances On / Off.
